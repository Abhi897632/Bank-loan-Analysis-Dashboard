# Bank-loan-Analysis-Dashboard
Interactive Power BI dashboard for exploring loan applications, funding, repayment status and customer segments with multi-level filters and drill-downs.

This project is a complete end-to-end Power BI report designed to explore and understand bank loan application data.
It includes multiple report pages, loan status segmentation visuals, funding distribution details, borrower-related breakdowns, and a full record-level drilldown section for deeper analysis.

Dashboard Pages Included
1. Summary Page:
Total Loan Applications,
Total Funded Amount,
Total Amount Received,
Average Interest Rate,
Average DTI,
Good vs Bad Loan Distribution,
Loan Status Table (Fully Paid, Current, Charged Off)

2. Overview Page:
Funded Amount by Month (Line Chart),
Funded Amount by State (Map Visual),
Funded Amount by Employee Length (Bar Chart),
Funded Amount by Purpose (Bar Chart),
Funded Amount by Home Ownership (Treemap),
Term-wise Loan Distribution (36 vs 60 months)

3. Details Page:
Full record-level data table including:
ID,
Purpose,
Home Ownership,
Grade & Sub-Grade,
Issue Date,
Funded Amount,
Interest Rate,
Installments,
Amount Received,
Supports complete filtering and drilldown for deep analysis.

Tech Stack & Skills Used
Power BI Desktop,
Power Query (Data Cleaning & ETL),
DAX (KPIs, Calculated Measures),
Data Modeling (Relationships, Star Schema),
Dashboard UI/UX
